Google Chrome 設定首頁教學

  1. 打開 Google Chrome 瀏覽器。
  2. 點擊右上角的三個點圖標 (設定選單)。
  3. 選擇 "設定"。
  4. 在左側菜單中選擇 "起始畫面"。
  5. 選擇 "開啟特定網頁或一組網頁"，然後點擊 "新增新網頁"。
  6. 輸入 http://192.168.1.136/welcome/ 並確認。

Microsoft Edge 設定首頁教學

  1. 打開 Microsoft Edge 瀏覽器。
  2. 點擊右上角的三個點圖標 (設定選單)。
  3. 選擇 "設定"。
  4. 在左側菜單中選擇 "開始、首頁及新索引標籤"。
  5. 選擇 "開啟這些頁面"，然後點擊 "新增頁面"。
  6. 輸入 http://192.168.1.136/welcome/ 並確認。
